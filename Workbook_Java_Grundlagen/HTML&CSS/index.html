Zugriffsrechte und Pakete (packages)

private
nur innerhalb der Klasse kann auf die Eigenschaft oder die Methode zugegriffen werden

protected
Zugriff ist innerhalb der Vererbungshierachie erlaubt
aber: in Java haben auch alle Klassen des gleichen Paketes Zugriff auf protected-Elemente

public
öffentlich, Zugriff ist von "überall" aus möglich
<none>
alle Klassen des gleichen Pakets haben Zugriff

Pakete

Pakete dienen dem strukturieren von Quellen und dem "Sauberhalten" des Namenraumes

Pakete / Klassen aus fremden Quellen können importiert werden

Alle Klassen aus einem Paket

import Paket.*;

eine Klasse aus einem Paket

import Paket.Klasse;

Pakete werden auf Verzeichnisstrukturen gemappt

Pakete dürfen weiter unterteilt werden (Subpakete)

um einen weltweiten Austausch von Paketen zu vereinfachen, verwendet man für Paketnamen üblicherweise Domainnamen
