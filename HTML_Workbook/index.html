Verlinkungen in HTML

zurück
  • auf der Webseite navigieren
  • auf andere Webseiten verweisen
  • Verknüpfung relativ zum Dokument
  • Beschriftung kurz und bündig
  • keine Sterotypen verwenden, wie z.B.: hier klicken mehr
  • Wort Link am Anfang vermeiden, wg. Screenreadern
  • Links nicht zu dicht nebeneinander
Einbindung:
<a href="index.html" title="Beschreibung">Home<a>
So sieht es aus:

Hinweis: Inlineelemente werden nie um Blockelemente verschachtelt!

FALSCH: <a href="index.html"><li>Home</li></a>
RICHTIG: <li><a href="index.html">Home</a></li>
Verlinkung mit sich selbst: <a href="#">Link auf die aktuelle Seite</a>
Verlinkung zu externen Seiten: <a href="http://www.google.de/" target="blank">google.de</a>
target: Ziel in dem die Seite geöffnet wird, blank = neues Fenster/Tab und self = im selben Fenster/Tab öffnen

Verlinkungen im gleichen Dokument

Anker (Ziel): <a id="Absatz1" ><h1>Absatz1</h1></a>
Link zu diesem Anker: <a href="#Absatz1" >Absatz1</a>

Hinweis: Das name-Attribut im a Tag ist nicht mehr W3C-Konform für HTML5!

Anweisungen: Schriftart, Schriftgröße, Hintergründe, Farben, Abmessungen, Rahmen...

CSS:
  • Trennung von Layout und Inhalt
  • mehr Formatierungsmöglichkeiten
  • schnelle Änderung aller Formate
  • einheitliche Darstellung für
  • unterschiedliche Geräte
Aufbau einer CSS-Regel:

Selektor{Eigenschaft:Wert; Eigenschaft2: Wert2; ...}
Selector{Property:Value; Property2: Value2; ...}

-Selektor steht vor der geschweiften Klammer
-wählt aus, was formatiert werden soll
-Deklaration -> Eigenschafts-Wert-Paar
-beschreibt die Gestaltung der Elemente

Eigenschaft (property):
-steht zuerst da (Farbe, Schriftart -> in englisch)

Wert (value):
-die zu gestaltene Eigenschaft
(Pixelangaben, %-Angaben; Farbangaben mit Farbwörtern oder Hexadezimalcode)

Kommentare:
/* Comment */

Einbindung:

CSS kann an 3 verschiedenen Stellen im Dokument eingebunden werden.
  1. CSS-Regel in der externen CSS-Datei *.css im head-Bereich mit Hilfe von <link>
    • <head>
      <link href="style.css" rel="stylesheet" type="text/css">
      </head>
  2. Zwischen <head> </head> mit Style-Element
    • <head>
      <style type="text/css">CSS-Regeln</style>
      </head>
  3. Direkt im HTML-Element (Inline-Stil)
    • <p style="font-size:3em;">Text</p>
Priorität:

Hinweis: die Regel die am dichtesten am Element (zuletzt deklariert) steht gilt!

So sieht es aus:

div{
border:1px solid black;
width:200px;
height:200px;
background:blue;
}

CSS Selektoren

zurück
  • Universalselektor, legt die Stilregeln für alle Elemente fest, wird für CSS Reset verwendet
    Beispiel: *{margin:0; padding:0;}
  • Tag- oder Typselektor, legt für ein einzelnes HTML-Element Stilregeln fest
    Beispiel: body{font-family:Veranda,Arial,sans-serif;}

    Hinweis: Schriftgrößen sollten immer relativ angegeben werden!

    Schlüsselwörter: small, medium, large
    festen Einheiten: Pixel (px)
    relative Einheiten: %, em
  • Klassenselektor
    • wird im HTML durch das Attribut class definiert
    • in css-Datei wird der Klassenselektor mit .Klassenname{Eigenschaft:Wert;} angelegt
    Beispiel: <p class="myparagraph">Text</p>

    Hinweis: Klassennamen selbst wählbar, keine Sonderzeichen oder Umlaute, darf nicht mit einer Ziffer beginnen, darf mehrfach verwendet werden.

  • ID-Selektor
    • wird im HTML mit dem Attribut id angelegt
    • darf nur einmal im Dokument vorkommen
    • frei wählbar (selbe Bedingung wie bei Klassennamen)
    • in der CSS-Datei wird der ID-Selektor mit der Raute eingeleitet, #id{Eigenschaft:Wert;}
    Beispiel: <div id="wrapper">Text</div>
  • Pseudoklassen: link, visited, hover, active, first-child, focus, lang
    Beispiel: <a href="#">Link</a>
    a:visited{color:red};

Abhängiger-Selektor

2 oder mehr Selektoren werden mit Leerzeichen nebeneinander geschrieben. Formatierung wird nur im 2. Element angewandt, wenn es im 1. steht.

Beispiel:
main h4{}
footer h4{}

Kind-Selektor

Zwei Selektoren werden mit > verbunden, 2. Element muss direktes Kindelement des 1. Elementes sein.
Beispiel:
main > article{}

Tiefen-Selektor

2 Selektoren werden mit * verbunden
2. Element muss mindestens 2 Ebenen tiefer liegen
Beispiel:
main * div{}

Geschwister-Selektor

2 Selektoren werden mit dem + verbunden.
2. Element liegt neben dem 1. Element und darf kein Kind-Element des 1. sein.
Beispiel: h6 + p{}

Attribut-Selektor

  • TAG-Name[attribut]{} /* Hier wird formatiert, wenn in einem Element das Attribut enthalten ist. */ Beispiel: a[title]{...}
  • TAG-Name[attribut='Wert']
    Beispiel: input[type='text']
  • TAG-Name[attribut^='wer'] Anfang des Wertes muss enthalten sein
  • TAG-Name[attribut$='ert'] Ende des Wertes muss enthalten sein
  • TAG-Name[attribut*='er'] Zuweisung muss innerhalb von Wert enthalten sein
